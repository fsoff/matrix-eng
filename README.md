# MSP Interactive Matrix (16×16)

This repository publishes an interactive MSP 16×16 interaction matrix via GitHub Pages.

## Live page
After enabling GitHub Pages, open:

https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/

(Serves `index.html`)

## Files
- `index.html` – interactive MSP matrix

## Enable GitHub Pages
1. Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: / (root)
5. Save
